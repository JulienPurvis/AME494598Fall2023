Temperature/Humidity

How to run server and watch sensors

Open your sanbox lab and star the lab.
Once the lab has started open the ec2 portal and start a new instance
Create the instance on ubuntu (20.04) with 28gb of storage and no key pair

Add inbound rules to the security group allowing inbound traffic anywhere ipv4 on ports 8080, 80, and 1234

Once the instance is created ssh into the instance and run the following commands
curl -s https://gist.githubusercontent.com/tejaswigowda/f289e9bff13d152876e8d4b3281142f8/raw/df37df2e16a3166e686357a045751a7c18bbeebe/installNodeMongoUbuntu16.04.sh | bash
sudo npm install -g n
sudo npm install -g latest

Clone the git repository into the EC2 instance with
Git clone “Repo link”
Navigate to the server directory on your repo and run the following
Node server.js

Now for the Watch:
Add the code from this link into a new sketch
Change the ssid and password to match your personal hotspot
Change the server url to match the public IP of your ec2
Navigate to the public ip of your server in your bro
Push the code to your watch and watch the magic happen
